## お支払いについて

スポンサー費用はHTML5 Conferenceなどのイベントの会計をしています日本オープン・ウェブ・アソシエーションからの請求になります。請求方法はスポンサー様ごとに異なるので、それに合わせますとのことです。開催にかかる費用は基本建て替えていただけるとのことです。請求書の期限はイベントの一ヶ月後あたりにしてくださるそうで、五月中に入金の必要があるとかではありません。

## スポンサー特典

* ホームページにリンク付のロゴが掲載されます
* 当日配布するイベントのパンフレットにロゴを掲載します
* [HTML5 Experts.jp](https://html5experts.jp/)に後日開催レポートが掲載され、こちらにリンク付きのロゴが掲載されます
  * こちら、一日で2,000-3,000PVになるそうです

> また「HTML5カンファレンス in 鹿児島」は、鹿児島で活動されている企業において、HTML5という最先端技術に対し積極的に係ることで、様々な産業への技術アピール、企業イメージなどをアピールすることができます。スポンサー様はこの活動にも間接的に参加していただいていることになり、地域貢献を行う企業イメージを、より一層高めることが期待できます。
